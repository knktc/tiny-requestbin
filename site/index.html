📨 Tiny RequestBin

A lightweight HTTP request inspector and debugging tool

Capture, inspect, and debug HTTP requests with ease

Open Web Interface View on GitHub

Features

🚀

Lightweight & Fast

Simple implementation with minimal dependencies. Written in Go for maximum performance.

🔍

Request Inspection

View detailed information about incoming HTTP requests including headers, body, and metadata.

🌐

Clean Web UI

Intuitive interface to browse and inspect captured requests with pagination support.

📋

CLI Mode

Option to print requests directly to the terminal for quick debugging.

🔒

Local Only

All data stays on your machine. No external services, no data leaks.

🐳

Docker Support

Multi-architecture Docker images available for easy deployment.

How It Works

1

Start the Server

Launch Tiny RequestBin using Docker, Go, or from source

tiny-requestbin -port 8282
2

Send Requests

Direct your HTTP requests to the server at any path

curl http://localhost:8282/test/path
3

Inspect & Debug

View captured requests in the web interface

Open http://localhost:8282/

Quick Start

🐳 Docker

docker run -p 8282:8282 knktc/tiny-requestbin

📦 Go Install

go install github.com/knktc/tiny-requestbin@latest

🔨 Build from Source

git clone https://github.com/knktc/tiny-requestbin.git
cd tiny-requestbin
go build

🔧 Webhook Development
